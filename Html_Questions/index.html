Question :

Whatâ€™s the difference between html tag and attribute and give example of some global attributes? (2 Marks)

Answer :

HTML tags are used to define the structure and layout of content within an HTML document. They are enclosed within angle brackets ("<" and ">") and usually come in pairs: an opening tag and a closing tag. Tags mark the beginning and end of HTML elements, and each tag serves a specific purpose in determining how the content within it should be rendered or behave.

HTML attributes provide additional information or properties to HTML elements. They are placed within the opening tag of an element and modify the element's behavior, appearance, or functionality. Attributes consist of a name and a value, separated by an equals sign ("="). Multiple attributes can be added to an element to provide different properties.

Global attributes, as the name suggests, can be applied to almost any HTML element. They are not specific to a particular element and can be used across multiple elements.
Example of global attributes:
  • class: Assigns one or more class names to an element for CSS and JavaScript purposes.
  • id: Provides a unique identifier to an element for targeting and manipulation.
  • style: Applies inline CSS styles to an element.
  • title: Specifies additional information about an element, usually displayed as a tooltip.
